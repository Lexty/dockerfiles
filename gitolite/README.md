# gitolite docker service
Based on [miracle2k's gitolite](https://github.com/miracle2k/dockerfiles/tree/master/gitolite).

## Usage
Build command:
```
make
```

Run command:
```
make run
```

Remove container command:
```
make kill
```

For shell connect to container:
```
make shell
```

Get ip of container:
```
make ip
```
